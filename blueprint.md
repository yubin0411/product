
# Lotto Number Generator

## Overview

This project is a simple web application that generates and displays random lottery numbers. It's built with HTML, CSS, and JavaScript, following modern web development practices.

## Implemented Features

*   **User Interface**: A clean and modern interface for displaying the lottery numbers and a button to generate new ones.
*   **Responsive Design**: The layout adapts to different screen sizes, ensuring a good experience on both desktop and mobile devices.
*   **Modern Styling**: Uses modern CSS features for a visually appealing design.

## Current Plan

*   **HTML Structure**: Create the basic HTML structure for the lottery number generator, including a title, a container for the numbers, and a "Generate" button.
*   **CSS Styling**: Style the elements to create a visually appealing and user-friendly interface. Use CSS variables for colors, flexbox for layout, and add some basic animations.
*   **JavaScript Logic**: Implement the functionality to generate 6 unique random numbers between 1 and 45 and display them on the screen when the "Generate" button is clicked.
